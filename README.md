# analisis-ventas-sql-server
Análisis de ventas usando SQL (consultas, KPIs y segmentación de Clientes)
Objetivo
Analizar el comportamiento de ventas de una empresa para identificar productos más vendidos, clientes clave y distribución de ingresos.
Herramientas utilizadas
SQL Server
SQL consultas

Resultados clave
- Total de ventas: 450
- Ventas por producto
- Ventas por cliente
- Participación porcentual por cliente. Cliente top: Juan (48.8% del total)

Insights
- El producto más vendido es "Polo"
- El cliente con mayor participación es Juan
- La ciudad con mayor volumen de ventas es Lima

  Conclusión
  Se recomienda enfocar estratégias comerciales en los productos de mayor rotación y en clientes con mayor participación
